# 🧗 Climb Tracker

A simple climbing session tracker built with React Native (Expo).

## 🎯 Purpose

A fast and minimal app for logging climbing sessions without interrupting your flow in the gym.

## ⚡ Features

- Start and end climbing sessions
- Quick climb logging (name, grade, result)
- Track attempts, sends, flashes, and onsights
- View session summary

## 🧱 Tech Stack

- React Native (Expo)
- TypeScript
- React Navigation
- Local state (MVP)

## 🚀 Getting Started

```bash
git clone https://github.com/your-username/climb-tracker.git
cd climb-tracker
npm install
npx expo start
